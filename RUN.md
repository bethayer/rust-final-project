### Prerequisites
- **Rust** (Install using rustup or follow the directions on the rust language website)
### Installation
1. Clone the repository:\
`git clone https://github.com/bethayer/rust-final-project`

2. Build using cargo:\
 `Cargo build`
3. Run using cargo:\
 `Cargo run --release`
4. Type: `train`
5. Our suggested parameters for training when prompted (~75% accuracy):
- Hidden layers: `2`
- Neurons: `128`
- Learning rate: `0.001`
- Epochs: `5`
6. Type: `Predict` for 1 test sample
7. `Quit` when finished
